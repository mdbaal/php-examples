## Wordpress Snippets

- <b>PluginBase</b> - Basis class voor een wordpress plugin. Deze wordt als eerste aangeroepen na de plugin_hook
- <b>SettingsManager</b> - Renderd settings en beheert het laden en opslaan daarvan.
